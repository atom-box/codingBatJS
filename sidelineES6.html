Top Ten ES6 Features

  1. Default Parameters These are written as EQUALS in the arguments. So...
    var link = function(height = 50, color = 'red', url = 'http://example.com') { ... }
  2. Template LiteralsYou no longer have to make a 'string sandwich' to insert data. JS understands let s = `Hello ${my-dude}.`
  3. Multi-line stringsThat same tick mark I put in the last card: "`" is also used for long quotes. I have used this to put the whole text of MacBeth in a single string.
